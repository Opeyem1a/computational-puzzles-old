The rules are that you have to colour the image in a way so that no two neighbouring regions have the same colour. That means, if you pick a line, there must be different colours on either side of the line. Check out the example below to see what we mean!

Example

Say we wanted to colour in this simple circle image in our special way.

The first attempt doesn't work, since some neighbouring regions have the same colours.

The second attempt, however, manages to colour the image without any of the colours touching one another.

You see how there are 5 regions in this simple circle image? That means we could have just picked 5 different colours and coloured the image in the special way. However, the second attempt used the minimum number of colours needed to colour this example image in our special way! This is a better solution because it saves resources.
